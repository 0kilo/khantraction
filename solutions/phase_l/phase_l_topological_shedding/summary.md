`omega_blindness_check.csv` is exactly flat. That means the current runtime is `omega`-blind by construction. It should not be interpreted as a first-principles derivation of masslessness.

`sample_packet_trajectory.csv` shows a Gaussian packet translated at fixed speed. That is a useful proxy visualization, but not a solved wave equation on a regenerated background.

1. the active runtime contains a full-domain phi-gated shedding proxy,
