q = omega + thetai + phij + rhok.

We raise the exponential funtion e to the power of q,
e^q = e^(omega + thetai + phij +rhok).

By exponential rule,
e^omega * e^(thetai) * e^(phij) * e^(rhok).

Then by Euler's,
e^omega * (cos(theta) + sin(theta))i * (cos(phi) + sin(phi))j * (cos(rho) + sin(rhok))k.

Let,
alpha = e^omega,
beta = cos(theta) + sin(theta)),
gamma = (cos(phi) + sin(phi)),
delta = (cos(rho) + sin(rhok)).

Domain
alpha: (-infinity, infinity)
beta, gamma and delta: [-2pi, 2pi] 

Hence alpha is always > 0 and e^q is O(4) group.
